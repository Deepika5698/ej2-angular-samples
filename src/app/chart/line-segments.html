In this example, you can see how to render and configure the points in a particular range by using MultiColoredLine series. Points under the range can be customized with Color and DashArrayproperties in the ChartSegment.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use multi colored line series feature, we need to inject MultiColoredLineSeriesService into the @NgModule.providers section.

More information about the line type series can be found in this documentation section.
