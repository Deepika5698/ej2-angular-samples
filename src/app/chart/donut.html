This Angular Donut Chart example visualizes the mobile browsers statistics by using doughnut series. Datalabels show information about the points.

In this example, you can see how to render and configure a doughnut chart. To create a doughnut in pie series, customize the InnerRadius property. DataLabels are used to represent individual data and its values.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch-enabled devices.

More information about the doughnut in pie series can be found in this documentation section.
